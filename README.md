# consumer360-data-analytics

## Project Description
Consumer360 is an internship-level data analytics project focused on analyzing customer behavior using SQL, Python, and Power BI.  
The goal of this project is to build a strong analytics foundation and enable data-driven decision making.

## Business Use Case
Businesses often struggle to understand customer behavior, identify churn risk, and analyze purchasing patterns.  
This project helps analyze customer data to gain meaningful insights and support business decisions.

## Project Scope
The project will be developed in multiple stages.

Current focus:
- SQL-based data analysis
- Data cleaning and transformation
- Basic exploratory analysis

Future scope:
- Advanced analytics
- Power BI dashboards
- Python-based analysis
